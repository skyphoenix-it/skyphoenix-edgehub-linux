Your second screen, native on Linux.

EdgeHub is a Rust + Qt 6/QML dashboard for the Corsair Xeneon Edge and other secondary or portrait displays, with a companion Manager for live layout and appearance editing.

No beta or release offer is live. The current branch is unreleased. Feature freeze, code freeze, performance certification, broad distro support, paid delivery and the final release gate are not complete. This page intentionally contains no checkout or download promise.

What exists today

Touch-first Hub

Swipeable multi-page layouts, portrait and landscape rendering, display targeting, hot-plug handling and on-device controls.

Desktop Manager

A standalone companion mirrors the layout and pushes reorder, resize and appearance changes to a running Hub over a local socket.

Local and inspectable

Plain local TOML configuration, no account requirement, no telemetry implementation, and one central egress gate for configured network features.

Layered verification

Rust, C++, QML, compositor-backed GUI, runtime, Manager and physical-hardware suites exist. The final immutable-candidate run is still required.

What remains before release

  • Close every release requirement and release-blocking defect.

  • Measure CPU, memory, startup and growth on the exact candidate.

  • Close legal, default, payment, refund, support and fulfilment decisions.
