One dashboard, two ways to shape it

Touch-first Hub

Swipeable multi-page layouts, portrait and landscape rendering, display targeting, hot-plug handling, and large on-device controls.

Live desktop Manager

Add screens, switch pages, arrange widgets, and change orientation or appearance from your main display. The running Hub follows over a local socket.

Local and inspectable

Plain local TOML configuration, no account requirement, no telemetry implementation, and one central egress gate for configured network features.

Native Linux stack

A Rust core handles configuration and metrics while Qt 6/QML renders the Hub and Manager. There is no browser shell or web server.
