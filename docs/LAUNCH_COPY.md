# EdgeHub launch-copy hold

**Do not publish.** The project is still alpha/development. No feature freeze,
code freeze, beta tag or release-ready candidate has been declared.

The previous Product Hunt, Show HN and Reddit drafts were removed because they
mixed current implementation facts with unsupported performance, package-store,
auto-update, paid-download, refund and support promises.

## Verified factual core for a future draft

> EdgeHub is a native Rust + Qt 6/QML dashboard for the Corsair Xeneon Edge and
> other secondary or portrait displays. The current development branch contains
> 30 first-party widgets, 19 preset screens, 29 themes, 29 accents and 10 animated
> backgrounds plus Gradient. A companion Manager edits layouts and appearance and
> pushes changes to a running Hub over a local socket. Configuration is stored
> locally; network-capable widgets use a central egress gate.

This paragraph describes the source tree only. Before publication it must be
paired with the actual release tag, supported-platform matrix, download routes
and measured candidate evidence.

- Advertised packages are published and independently install/upgrade/uninstall
  verified.

- Target-hardware CPU, RSS, startup and long-soak results are recorded.
- Legal review and the exact trademark disclaimer are approved.
- Any price, refund, support, payment and licence-delivery terms are real and
  tested.
- Fresh screenshots show the exact released candidate and are labelled honestly.
