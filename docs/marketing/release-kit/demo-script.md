# 60-second release demo

> Capture this from the exact signed candidate on the real Edge and verified
> desktop target. No simulated pointer, fake data, or edited UI state should be
> presented as live behavior.

## Capture specification

- 16:9 master, 2560×1440 or 1920×1080.
- Record the physical Edge and desktop Manager clearly enough to show which app
  owns each interaction.
- Use non-sensitive demo data and a clean temporary configuration.
- Keep the cursor and touch point visible where practical.
- Avoid Corsair logos and trade dress in designed overlays.
- End card must contain the independent-project disclaimer.
