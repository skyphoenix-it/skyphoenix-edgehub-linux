# Touch UI Review

Review without editing initially.

Check:

- Portrait and landscape layouts
- Fractional scaling
- Touch targets of approximately 48 logical pixels or larger
- Touch-only operation
- No hover-only functionality
- Scroll, drag, long press, cancellation, and gesture conflicts
- Accidental activation
- Text readability
- Loading, empty, disabled, disconnected, and error states
- Reduced-motion behavior
- Animation cost and input latency
- Keyboard and mouse fallback

Return prioritized findings with evidence and concrete corrections.
