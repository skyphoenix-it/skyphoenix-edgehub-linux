# Release Readiness

Review:

- Version consistency
- Changelog
- Build and packaging
- Arch/CachyOS package
- Ubuntu/Debian package
- Clean install and uninstall
- Configuration migration
- Upgrade and rollback
- Automated tests
- Hardware and manual UAT
- Portrait and landscape
- Wayland and X11
- Touch-only operation
- Display reconnect
- Suspend and resume
- Security findings
- Dependency audit
- Performance measurements
- Soak-test results
- Diagnostics and troubleshooting
- Documentation and known limitations

Classify the release as:

- Ready
- Ready with accepted risks
- Not ready

List blocking evidence still required.
